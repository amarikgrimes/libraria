📚 Libraria
A command-line library management system built with Python and SQLite. Libraria lets you manage books, authors, genres, members, and borrowing activity all from the terminal.

Features
- Add and manage books, authors, and genres
- Register library members
- Borrow and return books with tracking
- Run advanced queries (search by author, genre, availability, etc.)
- Lightweight local storage with SQLite — no server required
